# Mouse Controls

Complete reference for mouse controls in the MatterCAD 3D viewport.

## Navigation

| Action | Mouse Control |
|--------|--------------|
| **Rotate view** | Right-click + Drag |
| **Pan view** | Middle-click + Drag |
| **Zoom** | Scroll wheel |

## Alternative Navigation

| Action | Mouse Control |
|--------|--------------|
| **Rotate view** | Ctrl + Left-click + Drag |
| **Pan view** | Ctrl + Shift + Left-click + Drag |
| **Zoom** | Ctrl + Alt + Left-click + Drag |

## Object Interaction

| Action | Mouse Control |
|--------|--------------|
| **Select object** | Left-click on object |
| **Multi-select** | Shift + Left-click on additional objects |
| **Multi-select** | Ctrl + Left-click on additional objects |
| **Deselect all** | Left-click on empty space |
| **Rubber band select** | Left-click + Drag on empty space |
| **Move object** | Left-click + Drag on selected object |
| **Context menu** | Right-click on object |

## Object Controls

When an object is selected, interactive controls appear:

| Action | Control |
|--------|---------|
| **Move on bed** | Drag the object |
| **Move up/down** | Drag the vertical arrow above the object |
| **Rotate** | Click and drag a rotation corner control |
| **Scale** | Click and drag a scale corner control |

## Tips

- Zoom with the scroll wheel is centered on the mouse cursor position -- point at the area you want to zoom into
- The alternative controls (Ctrl + Left-click) are useful if you don't have a middle mouse button or right-click is inconvenient
- When rotating with corner controls, hover over the angle indicators to snap to 45-degree increments
