# Keyboard Shortcuts

Quick reference for all keyboard shortcuts available in MatterCAD.

## Edit

| Shortcut | Action |
|----------|--------|
| Ctrl + Z | Undo |
| Ctrl + Y | Redo |
| Ctrl + Shift + Z | Redo (alternative) |
| Ctrl + A | Select all objects |
| Ctrl + C | Copy selected objects |
| Ctrl + X | Cut selected objects |
| Ctrl + V | Paste objects |
| Space | Clear selection |
| Escape | Cancel current drag operation |
| Delete | Delete selected objects |

## View

| Shortcut | Action |
|----------|--------|
| W | Reset view (fit all objects in viewport) |
| Z | Zoom to selected object |
| Ctrl + Plus (+) | Zoom in |
| Ctrl + Minus (-) | Zoom out |
| Arrow Keys | Rotate the view |
| Shift + Arrow Keys | Pan the view |

## Mouse Controls

| Action | Control |
|--------|---------|
| Rotate view | Right-click + Drag |
| Pan view | Middle-click + Drag |
| Zoom | Scroll wheel |
| Select object | Left-click on object |
| Multi-select | Shift + Left-click or Ctrl + Left-click |
| Rubber band select | Left-click + Drag on empty space |
| Context menu | Right-click on object |

## Alternative Mouse Controls

| Action | Control |
|--------|---------|
| Rotate view | Ctrl + Left-click + Drag |
| Pan view | Ctrl + Shift + Left-click + Drag |
| Zoom | Ctrl + Alt + Left-click + Drag |

## Application

| Shortcut | Action |
|----------|--------|
| Ctrl + Shift + T | Reopen last closed design |
| F1 | Open Help |
