# Gear 2D

A 2D gear path that creates just the gear tooth profile as a flat path. Use this with [Linear Extrude](../operations/path/linear-extrude.md) or [Revolve](../operations/path/revolve.md) for more control over how the gear is built into a 3D shape.

![mechanical_gear_2d](https://matterhackers.github.io/MatterCAD_Docs/assets/mechanical_gear_2d.png)

## How to Use

1. Add a **Gear 2D** from the mechanical tools
2. Configure the tooth parameters
3. Apply [Linear Extrude](../operations/path/linear-extrude.md) to give it height

## Tips

- Use Gear 2D when you need to combine a gear profile with other path operations before extruding
- For a ready-to-use 3D gear, see [Gears](gears.md) instead
- The same meshing rules apply: matching Circular Pitch and Pressure Angle for gears that work together
