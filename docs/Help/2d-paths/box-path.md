# Box Path

A rectangular 2D path. Use with [Linear Extrude](../operations/path/linear-extrude.md) to create box shapes, or as a base for other path operations.

<!-- IMAGE_NEEDED: Screenshot of a Box Path on the workspace -->

## Parameters

- **Width** - The width of the rectangle (default: 20mm)
- **Height** - The height of the rectangle (default: 20mm)

## Tips

- A Box Path plus Linear Extrude creates a shape similar to a [Cube](../primitives/cube.md), but you can apply path operations like [Inflate](../operations/path/inflate-path.md) or [Smooth](../operations/path/smooth-path.md) before extruding
