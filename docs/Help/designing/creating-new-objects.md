## Undo and Redo  
![](https://www.matterhackers.com/r/Sp329R)
- Operations in MatterCAD become part of the undo history. Use these buttons or 'Ctrl + Z' and 'Ctrl + Y' to undo and redo actions
  
## Grouping and Ungrouping  
![](https://www.matterhackers.com/r/8Fq5aa)
- Grouping: Select multiple objects to group them together, allowing them to be moved and operated on as a single object
- Ungrouping: Grouped objects can be ungrouped at any time. This will also attempt to separate multiple bodies in a single STL if present
  
## Copying and Deleting  
![](https://www.matterhackers.com/r/KjnVvj)
- Use these buttons to copy or delete selected objects. System copy, cut, and paste features are also supported
  
## Align  
![](https://www.matterhackers.com/r/tIyb5N)
- Align multiple objects relative to each other with this tool
 
## Lay Flat  
![](https://www.matterhackers.com/r/a5kMJP)
- Quickly position a selection on the workspace as flat as possible by orienting the lowest face to the workspace plane
  
## Combine, Subtract, Intersect and Subtract & Replace  
![](https://www.matterhackers.com/r/cljE29)
- Create complex objects by combining simple shapes using these boolean operations

## Arrays  
![](https://www.matterhackers.com/r/NrrOSq)
- Create multiple copies of a part in an ordered and predictable pattern
  
## Transforms  
![](https://www.matterhackers.com/r/J5sl3b)
- Modify your parts with powerful transformation tools
