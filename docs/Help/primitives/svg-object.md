# SVG Object

Import SVG (Scalable Vector Graphics) files and use them as 2D paths in your design. SVGs can then be extruded into 3D shapes using [Linear Extrude](../operations/path/linear-extrude.md) or [Revolve](../operations/path/revolve.md).

## How to Use

1. Import an SVG file by dragging it onto the workspace or using the Open button
2. The SVG is imported as a 2D path
3. Apply [Linear Extrude](../operations/path/linear-extrude.md) to give it height, or use other [Path Operations](../operations/path/index.md)

## Tips

- SVG files should contain filled shapes or closed paths for best results
- Complex SVGs with many paths may take longer to process
- Use [Select Paths](../operations/path/select-paths.md) to work with specific parts of a multi-path SVG
- Many free SVG files are available online for logos, icons, and decorative patterns
