# Braille Card

Create Braille cards with embossed Braille text on an integrated base. This is a convenience tool that combines a Braille text object with a backing card in one step.

<!--  Screenshot of a Braille Card object showing dots on a rectangular card base -->
![20260318 182929 paste 20260318 182929](https://matterhackers.github.io/MatterCAD_Docs/assets/20260318-182929-paste-20260318-182929.jpg)


## How to Use

1. Add a **Braille Card** from the Primitives panel
2. Enter the text you want to encode
3. The tool generates both the Braille dots and a card base

## Tips

- Use this when you want a complete, ready-to-print Braille label
- For Braille dots without a base (to place on your own surface), use [Braille](braille.md) instead
