# Primitives

MatterCAD includes a library of 3D primitive shapes that serve as the building blocks for your designs. You can add any primitive to your workspace from the Primitives panel and then modify its parameters to get the exact shape you need.

## Basic Shapes

- [Cube](cube.md) - A box shape with adjustable width, depth, height, and optional rounding
- [Cylinder](cylinder.md) - A round column with configurable diameter, height, and sides
- [Sphere](sphere.md) - A round ball with adjustable diameter and detail level
- [Cone](cone.md) - A tapered cylinder that comes to a point
- [Torus](torus.md) - A donut shape with inner and outer diameter control
- [Ring](ring.md) - A tube shape (hollow cylinder) with inner and outer diameters
- [Pyramid](pyramid.md) - A four-sided pyramid with adjustable base and height
- [Wedge](wedge.md) - A triangular prism shape
- [Half Sphere](half-sphere.md) - The top half of a sphere
- [Half Cylinder](half-cylinder.md) - A cylinder cut in half lengthwise
- [Half Wedge](half-wedge.md) - A wedge cut in half
- [Hole](hole.md) - A cube-shaped hole used with boolean subtract operations

## Text and Special Shapes

- [Text](text.md) - Create 3D extruded text with customizable fonts and sizing
- [Braille](braille.md) - Generate 3D-printable Braille text
- [Braille Card](braille-card.md) - Create Braille cards with embossed text
- [QR Code](qr-code.md) - Generate QR codes as 3D objects
- [Image Object](image-object.md) - Import images and convert them to 3D relief
- [SVG Object](svg-object.md) - Import SVG vector graphics as 3D shapes
