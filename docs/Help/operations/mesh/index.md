# Mesh Operations
<!-- AUTO_IMAGE: type=toolbar_icons group=Mesh -->
![Mesh toolbar icons](https://matterhackers.github.io/MatterCAD_Docs/assets/toolbar-icons-Mesh.png)

Mesh operations help you clean up and optimize the triangle mesh geometry of your models.

## Operations

- [Repair](repair.md) - Fix problems in mesh geometry such as holes, non-manifold edges, and inverted faces
- [Decimate](decimate.md) - Reduce the polygon count of a mesh to simplify it while preserving shape
