# Outline Path

Outline Path creates an outline version of a filled path, converting a solid shape into just its boundary ring with a specified thickness.

<!-- IMAGE_NEEDED: Before and after showing a solid circle path converted to a ring outline -->

## How to Use

1. Select a 2D path
2. Apply **Outline Path** from the Path operations menu
3. Adjust the outline thickness

## Tips

- Use this to create frame-like shapes from solid paths
- Combine with [Linear Extrude](linear-extrude.md) to create 3D frames and borders
- An outlined [Circle Path](../../2d-paths/circle-path.md) is equivalent to a [Ring Path](../../2d-paths/ring-path.md)
