# Image Operations
<!-- AUTO_IMAGE: type=toolbar_icons group=Image -->
![Image toolbar icons](https://matterhackers.github.io/MatterCAD_Docs/assets/toolbar-icons-Image.png)

Image operations convert 2D images into 3D objects or paths, allowing you to incorporate photographs, logos, and artwork into your designs.

## Operations

- [Image Converter](image-converter.md) - Convert an image into a 3D relief based on brightness or color
- [Image to Path](image-to-path.md) - Trace an image to create a 2D path outline
- [Lithophane](lithophane.md) - Create a lithophane that reveals an image when backlit
