# What's New

* **All-New Direct3D Rendering Engine**
  * Dramatically faster and smoother 3D viewport powered by Direct3D 11
  * Correct transparency with dual depth peeling — see through overlapping objects accurately
  * Hardware-accelerated bed shadows and FXAA anti-aliasing for crisp, clean visuals
  * ![Screenshot of a design with one object set to 50% transparency, showing objects behind it](https://img.matterhackers.com/g/Z3M6Ly9taC1pcHMtcHJvZC9jbXMtcHJvZC80NGNmZGZlOC02NGI1LTRiZTEtYjE4Ni0wYTRhZjkxYWQxYzQ)

* **Object Transparency**
  * Set any object in your scene to be transparent to see what's inside or behind it
  * ![Show rotating a scene with multiple transparent objects and bed shadows](https://img.matterhackers.com/g/Z3M6Ly9taC1pcHMtcHJvZC9jbXMtcHJvZC9iNjNhOWMxNy00MzE0LTQ0ZmUtOGFjZS1kMWVlMTdkMzEzMDE)

* **Lock & Hide Objects**
  * Lock objects to prevent accidental selection or editing
  * Hide objects to reduce visual clutter while you work on specific parts
  * Quickly restore everything with Unhide All and Unlock All
  * ![Show locking an object (clicking lock icon), then trying to select it, then using Unlock All](https://www.matterhackers.com/r/g4j2gw)

## Improvements

* **Boolean Operations**
  * Multi-subtract operations are significantly more reliable
* **File Handling**
  * Projects now save as 3MF by default, preserving colors, materials, and design history
  * Enhanced drag-and-drop for files and folders directly into the 3D view
* **Workflow**
  * Save As and Move dialogs remember your last folder — no more drilling back from the root
  * Expression fields now support `pi`, `tau`, `e`, and `count`
* **Reliability**
  * Fixed startup and recursive load crashes
  * Improved library tree view updates
